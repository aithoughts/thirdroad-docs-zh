## Contracts

<div className="heroGrid">

<div>
  <p>部署智能合约是构建任何web3应用程序的第一步。</p>

你可以部署我们为更常见的用例(NFTs，tokens，marketplaces) [预先构建合约](/pre-built-contracts),
或者你可以使用我们的[Solidity SDK](/solidity)构建自己的智能合约，并通过[Deploy](/deploy)发布。

## Dashboard

<div className="heroGrid">
  
  <div>
    <p>
      你部署的每个合约都带有<a href='https://thirdweb.com/dashboard'>dashboard</a>来管理你的应用程序。
    </p>

这可以让你在没有任何代码的情况下为你的项目执行常见的管理操作，例如:

- **部署**新的智能合约，与当前合约进行**交互**
- **配置**版税和平台费用或其他合约设置
- 发布团队范围内的**权限**以启用协作
- 从智能合约分析中获取**洞察**

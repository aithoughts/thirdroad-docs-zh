### 开始之前

我们建议你在开始之前安装以下工具:

- [Node.js 和 npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm): 将我们的SDK安装到你的项目。
- [Yarn](https://classic.yarnpkg.com/en/docs/install/#mac-stable): 用于在React应用程序中执行脚本。
- [Git](https://git-scm.com/downloads): 在CLI中使用`create`命令。

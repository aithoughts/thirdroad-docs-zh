# CLI

CLI提供了使用thirdweb工具和基础设施创建、构建和部署项目的工具。使用CLI，你可以使用已配置的SDK创建新项目，构建、发布和部署智能合约，在开发时生成类型，将文件上传到IPFS等等

<OpenSourceInfo
  repoName="CLI"

  isLibrary={false}
/>

## 安装

全局安装CLI包，以便在系统的任何位置使用这些命令。

_注意：如果你不想全局安装它，你可以在每个命令前加上 `npx`。即 `npx thirdweb <command>`_。

<Tabs>
  <TabItem value="npm" label="npm" default>
    <CodeBlock language="bash">{`npm i -g @thirdweb-dev/cli`}</CodeBlock>
  </TabItem>
  <TabItem value="yarn" label="yarn">
    <CodeBlock language="bash">{`yarn global add @thirdweb-dev/cli`}</CodeBlock>
  </TabItem>
  <TabItem value="pnpm" label="pnpm">
    <CodeBlock language="bash">{`pnpm install -g @thirdweb-dev/cli`}</CodeBlock>
  </TabItem>
</Tabs>

安装完成后，你可以通过运行以下命令查看可用命令的完整列表：

```bash
thirdweb
```

## 可用命令

| 命令                        | 描述                                                                                           |
| --------------------------- | ----------------------------------------------------------------------------------------------|
| [`create`](/cli/create)     | 从我们的任何官方模板创建一个web3应用程序。 |
| [`install`](/cli/install)   | 将thirdweb的包添加到现有项目中。 |
| [`build`](/cli/build)       | 编译合约并检测thirdweb合约扩展。 |
| [`generate`](/cli/generate) | 获取合约ABIs，为SDK生成类型，提高性能。 |
| [`upload`](/cli/upload)     | 上传任何文件或目录到去中心化存储(IPFS)。 |
| [`deploy`](/deploy)         | 将你(或你的团队)的合约安全地部署到区块链上。 |
| [`publish`](/publish)       | 发布你的协议，这样其他开发者就可以部署它们，解锁SDKs、dashboard和analytics。 |
